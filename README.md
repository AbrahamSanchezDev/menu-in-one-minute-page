# 🏖️ Playa Azul Restaurant Menu Prototype

A lighting-fast **1-2 minute prototype** built as a sales pitch for a local restaurant in Playa Azul.

### 💡 The Pitch

The client felt developing a full-scale, multi-page website was "too much" and too complex for their current needs. To overcome this friction, this prototype was spun up instantly to prove that they don't need an expensive or heavy system to get started. They can establish a digital footprint immediately just by hosting their existing menu graphics on a responsive, zero-maintenance landing page.

### 🚀 Key Features

- **Zero-Cost Infrastructure:** Designed to be hosted completely free on **GitHub Pages**.
- **Smart Orientation Handling:**
  - 📱 **Portrait Mode (Mobile):** Stacks menus vertically at full screen width (`100%`) for a familiar, natural scroll experience on smartphones.
  - 💻 **Landscape Mode:** Locks into a clean, side-by-side view using dynamic viewport units (`100dvh`) to completely eliminate mobile browser address-bar clipping.
- **Instant Delivery:** Conceptualized, coded, and deployed in under 2 minutes to close the deal.

### 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Orientation Media Queries)
